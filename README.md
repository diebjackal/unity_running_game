# unity_running_game

유니티 기본기를 익히기 위해 간단한 러닝 게임을 만들었습니다

# 사용 중인 유니티 버전
 2019.4.18f1

# 기능

1. 코인을 먹으면 점수가 쌓입니다.
2. 필드에 있는 적의 머리를 밣으면 적을 죽일 수 있습니다.
3. 적에게 3번 공격을 받으면 죽게 됩니다.
4. 마지막 깃발에 닿게 되면 다음 스테이지로 가게 됩니다.
